# Land Use / Land Cover (LULC) Classification (2004, 2014, 2024)

This repository contains Google Earth Engine (GEE) scripts and outputs for multi-temporal Land Use/Land Cover (LULC) classification using the Random Forest classifier for the years 2004, 2014, and 2024. The workflow demonstrates a consistent classification framework to analyse decadal land transformation.

# Project Overview
Platform: Google Earth Engine (JavaScript API)
Method: Supervised Classification using Random Forest
Input Data: Landsat (2004 & 2014), Sentinel/Landsat (2024)

# Key Steps:
1. Preprocessing (cloud masking, mosaicking)
2. Training data preparation
3. Random Forest classification
4. Accuracy assessment (confusion matrix & overall accuracy)
5. Export of classified maps
